Funciones y Módulos

En Python, las funciones te permiten encapsular código reutilizable. Además, puedes importar módulos para ampliar las funcionalidades del lenguaje. Por ejemplo, el módulo math ofrece funciones matemáticas avanzadas.

Una función se define usando la palabra reservada def y puede devolver valores mediante return.

Entrada y Salida de Datos

Python permite interactuar con el usuario mediante la función input() para recibir datos y print() para mostrar información en pantalla, lo que facilita la creación de programas interactivos.

Ejemplo:

# Solicitar el nombre del usuario
nombre = input("Ingresa tu nombre: ")
print("Bienvenido, " + nombre)
