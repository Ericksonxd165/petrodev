Cálculo de Volumen de Reservas

En ingeniería petrolera, el volumen original de petróleo en sitio (OOIP) se calcula con la ecuación:

def calcular_OOIP(area, espesor, porosidad, factor_volumetrico):
    return (area * espesor * porosidad) / factor_volumetrico

# Ejemplo de cálculo de reservas
ooip = calcular_OOIP(5000, 10, 0.2, 1.25)
print(ooip) # Salida: 80000 barriles

Simulación de Flujo en Tuberías

En la ingeniería petrolera, se requiere calcular la velocidad del flujo en tuberías:

def calcular_velocidad_flujo(caudal, diametro):
    return (caudal / (3.1416 * (diametro/2) ** 2))

# Ejemplo
velocidad = calcular_velocidad_flujo(500, 8)
print(velocidad) # Salida: 9.95 ft/s

Cálculo del Índice de Productividad

El índice de productividad (IP) mide la eficiencia de un pozo para producir fluidos:

def calcular_IP(flujo, presion_fondo, presion_reservorio):
    return flujo / (presion_reservorio - presion_fondo)

# Ejemplo
IP = calcular_IP(500, 2500, 4000)
print(IP) # Salida: 0.33 stb/d/psi

Optimización de Producción con Funciones

En la ingeniería petrolera, optimizar la producción de un pozo requiere ajustar variables clave para maximizar la eficiencia. A continuación, una función que evalúa la producción óptima con base en el índice de productividad (IP) y la diferencia de presión:

def optimizar_produccion(IP, presion_reservorio, presion_fondo):
    flujo_optimo = IP * (presion_reservorio - presion_fondo)
    return flujo_optimo

# Ejemplo de optimización
flujo_optimo = optimizar_produccion(0.33, 4000, 2500)
print(flujo_optimo) # Salida: 495 stb/d
